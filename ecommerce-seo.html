üîç SEO Implementation Reference

Meta Tags in <head>

  • Optimized <title> tag (55 chars, keyword-rich)
  • Meta description (155 chars with CTA)
  • Meta keywords, author, and robots directives
  • Canonical URL to prevent duplicate content
  • Open Graph tags for social sharing previews
  • Twitter Card tags for Twitter/X previews

Structured Heading Hierarchy

  • H1: Site brand name ‚Äî one per page only
  • H2: Hero, Categories, Products, Promo, Features, Reviews, Newsletter, Footer cols
  • H3: Individual category, product, feature, footer column headings
  • H4: Reviewer names inside testimonials
  • No heading levels skipped (H1‚ÜíH2‚ÜíH3‚ÜíH4)

Structured Data (JSON-LD)

  • WebSite schema with SearchAction (sitelinks searchbox)
  • Organization schema with logo, contact, and social profiles
  • ItemList schema for Featured Products section
  • Product schema (Microdata) on each product card
  • Offer, AggregateRating, and Review schemas

Semantic HTML & Accessibility

  • <header>, <main>, <footer>, <aside> landmarks
  • <nav> with descriptive aria-label attributes
  • <article> for product and testimonial cards
  • ARIA labels on all interactive elements
  • Skip-to-content link for screen readers
  • Lang="en" on <html>, charset UTF-8

Performance & Crawlability

  • Mobile-responsive viewport meta tag
  • Descriptive alt text / aria-label on all images
  • Semantic <form role="search"> for site search
  • Internal navigation with descriptive anchor text
  • revisit-after meta for crawl scheduling

Content SEO Best Practices

  • Keyword-rich headings (fashion, electronics, deals)
  • Descriptive product names and descriptions
  • User-generated content signal (review counts)
  • Breadcrumb-ready category labelling
  • Clear CTAs that reinforce intent keywords
